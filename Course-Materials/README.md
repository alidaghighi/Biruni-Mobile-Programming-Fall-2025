In this course we start from the essentials of JavaScript and ES6 features, then we move on to ReactJS fundamentals, and finally we cover React Native for mobile application development. Each section includes theoretical explanations, practical examples, and hands-on exercises to reinforce learning.

This directory is only for course materials. Assignments and project files are located in their respective directories.

The course materials are organized as follows:

- [JavaScript ve ES6 Temelleri](JavaScript/README.md): This section covers the basics of JavaScript programming language, including variables, data types, functions, control structures, and ES6 features such as arrow functions, template literals, destructuring, and modules.

- [React Temelleri](React/README.md): This section introduces the core concepts of React, including components, props, state, and lifecycle methods. It also covers advanced topics such as hooks and context API.

- [React Native Temelleri](React-Native/README.md): This section focuses on building mobile applications using React Native. It covers the basics of mobile development, including navigation, styling, and accessing device features.
